Aim :- Attach EBS to ec2 server and create partitions and mount it on directory 

Create an ec2 instance ubuntu server
1) Create 100Gib volume in EBS services then select the volume and attached to the existing ec2 server machine.

2) ssh to the server check the volume is attached or not 
lsblk :- this command show attached volume to the server 

3) To create partition of volume the command is 
(fdisk /dev/nvme1n1) 
(n) for new partition
(p) primary partition
enter
enter
(+20G) it creates 20gb partition form the 100Gib volume that we attached to the instance
 run again lsblk you get the partion name and partition configurations

4) after creating partition 

mkfs.ext4 /dev/nvme1n1p1 this command make file system for our partition its like flash, or format the disk before mount on any directory.

5) (mkdir /backup) create directory to attached this directory to the partition that we created in earlier step

6) (mount /dev/nvme1n1p1 /backup) this command mount to the directory that we created for mount /backup.

7) (df -hT) mounted device configuration

8) edit /etc/fstab for permanent mount this mount will remains mounted after restarting the servers 

run blkid and copy nvme1n1p1 UUID 
copy UUID , go to vim
vim /etc/fstab 
UUID"        " .ext4 /root/backup defaults 0 0
